## คำอธิบายแผนภาพบริบท (Context Diagram Description)

ระบบบริหารจัดการและติดตามสถานะพาเลท (Pallet Management and Tracking System) มีการแลกเปลี่ยนข้อมูลกับ External Entities ดังนี้:

### 1. กลุ่มเจ้าหน้าที่ (Staff)
*   **Input (เข้าสู่ระบบ):** ส่งข้อมูล `ข้อมูลเข้าสู่ระบบ` (Username/Password) เพื่อยืนยันตัวตน
*   **Input (ทำรายการ):** ส่ง `ข้อมูลเบิก-คืนพาเลท` และ `ข้อมูลแจ้งพาเลทชำรุด` เพื่ออัปเดตสถานะพาเลท
*   **Input (ดูข้อมูล):** ส่ง `คำขอดูประวัติ (ค้นหา/กรอง)` เพื่อเรียกดูข้อมูลการใช้งานของตนเอง
*   **Output:** ระบบตอบกลับด้วย `ผลการตรวจสอบสิทธิ์`, `ผลการทำรายการ`, และ `ข้อมูลประวัติธุรกรรม`

### 2. กลุ่มผู้ดูแลระบบ (Admin)
*   **Input (เข้าสู่ระบบ):** ส่ง `ข้อมูลเข้าสู่ระบบ` เพื่อยืนยันตัวตนเข้าใช้งานส่วนจัดการ
*   **Input (จัดการข้อมูลหลัก):** ส่งข้อมูลเพื่อบริหารจัดการระบบ ได้แก่ `ข้อมูลจัดการพาเลท`, `ข้อมูลจัดการผู้ใช้งาน`, `ข้อมูลจัดการแผนก/สถานที่`
*   **Input (จัดการธุรกรรม):** ส่ง `ข้อมูลจัดการประวัติธุรกรรม` เพื่อแก้ไขหรือลบรายการที่ผิดพลาด
*   **Input (รายงาน):** ส่ง `คำขอดูรายงาน / ส่งออก` เพื่อดึงข้อมูลสรุป
*   **Input (ตั้งค่า):** ส่ง `ข้อมูลการตั้งค่าระบบ` เพื่อปรับเปลี่ยนพารามิเตอร์การทำงาน
*   **Input (ความปลอดภัย):** ส่ง `คำขอรีเซ็ตรหัสผ่าน` กรณีลืมรหัสผ่าน
*   **Output:** ระบบตอบกลับด้วย `ผลการตรวจสอบสิทธิ์`, `ผลการทำรายการ / จัดการข้อมูล`, `รายงาน Dashboard / ไฟล์ CSV`, และ `สถานะการส่งอีเมลรีเซ็ตรหัสผ่าน`

### 3. ระบบภายนอก (External Systems)
*   **LINE Messaging API:**
    *   **Input:** รับ `ข้อความแจ้งเตือน (Flex Message / Daily Report)` จากระบบเพื่อส่งต่อไปยังผู้ใช้
    *   **Output:** ส่งคืน `สถานะการส่งข้อความ (Response)`
*   **ผู้ให้บริการอีเมล (Email Service):**
    *   **Input:** รับ `ข้อมูลคำขอรีเซ็ตรหัสผ่าน` เพื่อส่งอีเมลไปยังผู้ใช้
    *   **Output:** ส่งคืน `สถานะการส่งอีเมล` ยืนยันการส่ง
*   **ตัวตั้งเวลาอัตโนมัติ (Cron Job / Scheduler):**
    *   **Input:** ส่ง `สัญญาณเรียกทำงานตามเวลา (Trigger)` เพื่อกระตุ้นระบบ (เช่น การสรุปยอดรายวัน)
    *   **Output:** รับ `สถานะการทำงาน (สำเร็จ/ล้มเหลว)` จากระบบเพื่อบันทึก Log การทำงาน
