# 3.10 ระบบเครื่องและอุปกรณ์ที่ใช้ในการพัฒนา

ในการพัฒนาระบบผู้พัฒนาได้ใช้เครื่องคอมพิวเตอร์ที่มีรายละเอียดคุณลักษณะเฉพาะ (Specifications) เพื่อรองรับการเขียนโปรแกรมและการจำลองระบบฐานข้อมูล ดังนี้

### **3.10.1 เครื่องคอมพิวเตอร์ประมวลผล (Hardware Specification)**
1)  **รุ่นเครื่อง**: Acer Nitro V 15 (ANV15-51)
2)  **หน่วยประมวลผลกลาง (CPU)**: 13th Gen Intel(R) Core(TM) i5-13420H ความเร็ว 2.10 GHz
3)  **หน่วยความจำหลัก (RAM)**: 16.0 GB
4)  **ระบบปฏิบัติการ (OS)**: Windows 11 Home 64-bit (Version 25H2)

### **3.10.2 อุปกรณ์ประกอบอื่น ๆ (Peripherals)**
1)  **สมาร์ทโฟน**: สำหรับใช้ทดสอบการสแกน QR Code ผ่านเว็บแอปพลิเคชัน
2)  **เครื่องพิมพ์ (Printer)**: สำหรับทดสอบการพิมพ์รหัสประจำตัวพาเลท

# 3.11 โปรแกรมทั้งหมดที่ใช้ในการพัฒนา

การพัฒนาระบบใช้เทคโนโลยี Modern Web Tech Stack เพื่อประสิทธิภาพและความรวดเร็ว ดังนี้:

### **1. โปรแกรมหลักและภาษาที่ใช้พัฒนา (Core Stack)**
*   **Node.js (v22.x)**: สภาพแวดล้อมหลักในการรัน JavaScript และจัดการเครื่องมือพัฒนา
*   **TypeScript (v5.8.2)**: ภาษาหลักที่ใช้เขียนโปรแกรมเพื่อลดข้อผิดพลาดด้วยระบบ Static Typing
*   **React (v19.2.1)**: ไลบรารีหลักสำหรับสร้างส่วนหน้าจอผู้ใช้งาน (UI) แบบ Component-based

### **2. ระบบจัดการหลังบ้านและฐานข้อมูล (Backend & Database)**
*   **Supabase (BaaS)**: บริหารจัดการฐานข้อมูล PostgreSQL, ระบบยืนยันตัวตน และพื้นที่เก็บไฟล์หลักฐาน
*   **Tailwind CSS (v4.1.18)**: เฟรมเวิร์กสำหรับออกแบบ UI ให้รองรับ Responsive Design
*   **Lucide React (v0.556.0)**: ชุดไอคอนมาตรฐานสำหรับแสดงผลในแอปพลิเคชัน

### **3. เครื่องมือช่วยพัฒนา (Development Tools)**
*   **Vite (v6.2.0)**: เครื่องมือ Build Tool สมรรถนะสูงเพื่อการดูผลลัพธ์แบบ Real-time (HMR)
*   **Visual Studio Code**: โปรแกรมหลักในการเขียนรหัสโปรแกรม (Coding Editor)
*   **Git & GitHub**: ระบบควบคุมเวอร์ชัน (Version Control) และพื้นที่จัดเก็บซอร์สโค้ด
*   **Chrome DevTools**: เครื่องมือสำหรับตรวจสอบและแก้จุดบกพร่อง (Debug) บนเบราว์เซอร์

### **4. ไลบรารีเฉพาะทาง (Specialized Libraries)**
*   **Html5-qrcode (v2.3.8)**: ไลบรารีสำหรับเข้าถึงกล้องและประมวลผลการสแกน QR Code ของพาเลทแต่ละตัว
